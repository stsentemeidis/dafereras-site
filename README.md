# Dr Georgios Dafereras — Site web

Site vitrine pour Dr Georgios Dafereras, Gynécologue-Obstétricien à Lausanne.

## Structure

- `index.html` — page unique (FR/EN)
- `css/style.css` — styles
- `js/script.js` — interactions (langue, menu mobile, formulaire)
- `images/` — médias

## Déploiement

Ce site est statique (HTML/CSS/JS) et peut être hébergé gratuitement sur GitHub Pages.
